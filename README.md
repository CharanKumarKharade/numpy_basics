# NumPy Basics Tutorial

This repository is a beginner-friendly NumPy tutorial covering the fundamentals and commonly used features of the NumPy library. It is useful for quick revision and interview preparation.

## What is NumPy?

[NumPy](https://numpy.org/) (Numerical Python) is an open-source library for scientific computing in Python. It provides support for large, multi-dimensional arrays and matrices, along with a collection of high-level mathematical functions to operate on these arrays efficiently.

## Topics Covered

- **Array Creation** – Creating arrays using `np.array()`, `np.zeros()`, `np.ones()`, `np.arange()`, `np.linspace()`, etc.
- **Array Indexing & Slicing** – Accessing and modifying elements using indices and slices.
- **Array Shape & Reshaping** – Understanding `shape`, `reshape()`, `flatten()`, and `transpose()`.
- **Mathematical Operations** – Element-wise operations, broadcasting, dot product, and more.
- **Aggregate Functions** – `sum()`, `mean()`, `min()`, `max()`, `std()`, `var()`, etc.
- **Boolean Indexing & Filtering** – Using conditions to filter array elements.
- **Random Module** – Generating random numbers with `np.random`.
- **Linear Algebra** – Matrix operations using `np.linalg`.
- **Universal Functions (ufuncs)** – Fast element-wise operations like `np.sqrt()`, `np.exp()`, `np.log()`.

## Why NumPy?

- Fast and memory-efficient array operations (backed by C under the hood)
- Foundation for popular libraries like Pandas, Matplotlib, TensorFlow, and scikit-learn
- Essential for data science, machine learning, and scientific computing
